# Icon 图标

---

### 介绍

图标组件

### 支持框架

`Vue3`、`Taro`

### 使用

```js
import { createApp } from 'vue';

// vue
import { Icon } from '@vb-design/vb-ui';
// taro
import { Icon } from '@vb-design/vb-ui-taro';

const app = createApp();
app.use(Icon);
```
